# Dollar Deal Club v2

This repository contains the Dollar Deal Club v2 application — a Next.js app with Supabase backend and Stripe billing integration.

For full documentation and deployment instructions see the `docs/` folder:

- Documentation root: [docs/README.md](docs/README.md)
- Vercel deployment instructions: [docs/vercel-deployment-setup.md](docs/vercel-deployment-setup.md)

Quick start
----------

Install and run locally:

```bash
npm install
npm run dev
```

Environment
-----------
All secrets are stored in `.env` locally and in your deployment provider (Vercel/GitHub Secrets). Do NOT commit secrets to Git.
